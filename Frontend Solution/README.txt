Frontend-only solution to the code challenge

Simply run this program by opening the "index.html" file

**Due to browser security, an extension to enable cross domain ajax request has to be installed for this program to work. The chrome extension can be gotten here: https://chrome.google.com/webstore/detail/cors-toggle/omcncfnpmcabckcddookmnajignpffnh

Tested and works on Google Chrome (with the installed extension).
